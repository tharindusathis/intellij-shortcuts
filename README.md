## Quick Navigation

| Key-binding                         | Action                                         | Preview                                                |
| ----------------------------------- | ---------------------------------------------- | ------------------------------------------------------ |
| `Ctrl + N`<br>                      | Navigate to a class                            | ![navigate-to-class](assets/img/navigate-to-class.gif) |

| `Shift + Ctrl + Alt + N`<br>`⌥ ⌘ O` | Navigate to a symbol                           |                                                        |
| `Shift Shift`                       | Search Everywhere                              |
